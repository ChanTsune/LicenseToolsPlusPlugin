# LicenseToolsPluginPlus

## Setup

```groovy
buildscript {
    dependencies {
        classpath "com.github.chantsune:license-tools-plus:${version}"
    }
}
```

```groovy
plugin {
    id "com.github.chantsune.license-tools-plus"
}
```
